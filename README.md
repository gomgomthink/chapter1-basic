# 🎓 학원관리 시스템

구글 드라이브를 백엔드로 활용하는 현대적인 웹 기반 학원관리 프로그램입니다.

## ✨ 주요 기능

### 🔐 로그인 시스템
- 선생님별 개별 로그인
- Google 계정 연동 로그인
- 오프라인 모드 지원

### 👥 학생 관리
- 학생 등록, 수정, 삭제
- 학년별 분류
- 재원/휴원/퇴원 상태 관리
- 학생 검색 및 필터링

### 📊 대시보드
- 전체 학생 수 현황
- 재원생/휴퇴원생 통계
- 오늘 수업 일정
- 최근 활동 내역

### 📅 시간표 관리
- 요일별 시간표 생성
- 강사 배정
- 학생별 수업 배정
- 시각적 그리드 뷰

### 📝 수업 기록
- 단어시험 점수 기록
- 듣기시험 점수 기록
- 숙제 완성도 체크
- 수업 메모 작성

### 📈 성적 관리
- 학교 시험 성적 입력
- 시험 종류별 분류 (중간고사, 기말고사, 쪽지시험)
- 과목별 성적 관리
- 학생별 성적 조회

### 📞 연락처 관리
- 학부모 연락처 관리
- 학생 연락처 관리
- 카드 형태의 직관적 UI

## 🚀 시작하기

### 1. 파일 다운로드
프로젝트 파일들을 로컬 컴퓨터에 다운로드합니다.

### 2. Google API 설정 (선택사항)
Google Drive 연동을 원할 경우:

1. [Google Cloud Console](https://console.cloud.google.com/)에 접속
2. 새 프로젝트 생성 또는 기존 프로젝트 선택
3. Google Drive API 활성화
4. OAuth 2.0 클라이언트 ID 생성
5. `google-drive-api.js` 파일에서 다음 부분 수정:
   ```javascript
   apiKey: 'YOUR_API_KEY', // 실제 API 키로 교체
   clientId: 'YOUR_CLIENT_ID', // 실제 클라이언트 ID로 교체
   ```

### 3. 웹 서버 실행
- 로컬 웹 서버에서 실행 (Live Server 등)
- 또는 웹 호스팅 서비스에 업로드

### 4. 첫 로그인
기본 계정 정보:
- **선생님 ID**: `teacher1`, **비밀번호**: `1234`
- **선생님 ID**: `teacher2`, **비밀번호**: `1234`

## 💻 사용법

### 로그인
1. 선생님 ID와 비밀번호 입력
2. 또는 Google 계정으로 로그인

### 학생 등록
1. 사이드바에서 "학생관리" 클릭
2. "학생 등록" 버튼 클릭
3. 학생 정보 입력 후 저장

### 시간표 작성
1. "시간표" 메뉴 선택
2. 빈 시간대의 "+" 버튼 클릭
3. 수업 정보 입력 후 저장

### 수업 기록 작성
1. "수업기록" 메뉴 선택
2. "기록 추가" 버튼 클릭
3. 학생, 점수, 메모 등 입력

### 성적 입력
1. "성적관리" 메뉴 선택
2. "성적 입력" 버튼 클릭
3. 시험 정보 및 점수 입력

## 🔧 기술 스택
- **프론트엔드**: HTML5, CSS3, JavaScript (ES6+)
- **백엔드**: Google Drive API
- **스타일링**: 현대적인 CSS Grid/Flexbox
- **아이콘**: Font Awesome
- **데이터**: JSON 형태로 저장

## 🆘 문제 해결

### Google Drive 연동 오류
1. API 키와 클라이언트 ID 확인
2. OAuth 동의 화면 설정 확인
3. 브라우저 쿠키/캐시 삭제

### 데이터 손실 시
1. 로컬 스토리지 백업 확인
2. Google Drive에서 수동 복구
3. 브라우저 개발자 도구에서 localStorage 확인

### 성능 최적화
1. 불필요한 브라우저 확장 프로그램 비활성화
2. 브라우저 캐시 정리
3. 안정적인 인터넷 연결 확인
